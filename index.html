Projects

Turret Trajectory & Laser Reflect

A very useful assignment that I did in my Game Development class that made me fully undestand Vector.normalized and Raycasts in Unity.

The Projectile Turret currently shoots a projectile in an arc toward a target point.

The Laser Turret fires a laser at a target point, but now the laser can bounce off surfaces. It stops at the next surface it hits, with optional recursion for additional bounces.

Completing these turrets required problem-solving and translating equations into code. For the trajectory turret, kinematic equations were used to calculate motion over time. For the laser turret, vector math and dot products were implemented manually (without Unity's Vector3.Reflect) to reflect the laser correctly.

Boss Fight

From my final project in game development, I created a boss fight for a third-person action/adventure game. The project uses a pre-existing game template, and namespaces are implemented to simulate working in a collaborative development environment.

This project involves implementing a state machine (or behavior tree) outside of the Enum State Machine Method that I was used to. The boss has three phases—Initial, Mid-Battle, and Enraged—each adding new attacks and mechanics as health decreases.

Required features include melee and ranged attacks, AoE or multi-stage attacks, telegraphed movements, environmental interactions, and defensive maneuvers. The project emphasizes problem-solving, translating game design into code, and implementing additional attacks and phases while integrating with existing systems.

This project demonstrates my ability to work with complex game systems, adhere to project constraints, and develop gameplay mechanics that interact correctly with existing code.
